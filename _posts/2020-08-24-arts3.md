   - window.location.origin不兼容IE，使用以下方法或者用其他属性（host、href等）兼容替代

   ```js
   if (!window.location.origin) {
      window.location.origin = window.location.protocol + "//" + window.location.hostname + (window.location.port ? ':' + window.location.port: '');
    }
   ```
